# puppygirlfastfetchascii
Just an upload of my personal ascii art i use for fast fetch

if i recall correctly the ascii art is html ascii but i dont remember exactly how it works tbh

just download the "fastfetch" folder and put it in your .config folder in your home directory! :)

NOTE:obviously you need fastfetch to use the fastfetch config files. it should be in most distros repositories. 
example
apt-get fastfetch
pacman -S fastfetch
dnf install fastfetch
